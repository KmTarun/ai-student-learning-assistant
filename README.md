# AI-Based Student Learning & Task Management Assistant

## Project Overview
This project is an AI-powered learning assistant designed to help students organize assignments,
plan exam preparation, and access relevant study materials efficiently.

## SDG Alignment
- SDG 4: Quality Education
- SDG 3: Good Health & Well-being

## Key Features
- Personalized student profile setup
- AI-based assignment task breakdown
- Exam preparation planning
- Intelligent content recommendation (RAG)
- Question and flashcard generation
- Weak area analysis

## Technologies Used
- Retrieval-Augmented Generation (RAG)
- Agentic AI
- IBM Granite Foundation Models
- Prompt Engineering
- Natural Language Processing (NLP)

## Project Status
This repository contains a conceptual prototype and UI mockup developed as part of the
1M1B AI for Sustainability Internship.
